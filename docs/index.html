The Architecture

Three layers of structured autonomy

A hierarchical system that gives AI agents clear boundaries and purpose.

Condos

Workspaces

Isolated environments where related work lives. Each condo has its own context and files.

Goals

Objectives

Discrete outcomes to achieve. Goals have status, priority, tasks, and deadlines.

Sessions

Conversations

Active agent threads doing work. Sessions execute goals and spawn subagents.

Capabilities

Everything agents need to stay on track

Built for AI-native workflows where context is currency.

Hierarchical Context

Goals inherit condo context. Sessions inherit goal context. Every agent knows its purpose.

Real-time Status

Live dashboards show active sessions, goal progress, and system health at a glance.

Apps Platform

Embed any web app with an AI assistant sidebar. Node.js, Python, React â€” anything.

Subagent Spawning

Sessions spawn child sessions for parallel work. Parent agents stay informed.

Voice & Media

Voice recording with Whisper transcription, image uploads, and media attachments.

File-backed State

All data persists as markdown files. Version control friendly. No database lock-in.

Organize Wizard

AI-assisted session triage. Automatically suggest goals and categorize conversations.

WebSocket API

Real-time API for agents to create goals, update progress, and spawn sessions.
