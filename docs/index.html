The Architecture

Three layers of structured autonomy

A hierarchical system that gives AI agents clear boundaries and purpose.

Condos

Workspaces

Isolated environments where related work lives. Each condo has its own context, files, and purpose.

Goals

Objectives

Discrete outcomes to achieve. Goals have status, priority, tasks, and deadlines.

Sessions

Conversations

Active agent threads doing work. Sessions execute goals and spawn subagents.

Capabilities

Everything agents need to stay on track

Built for AI-native workflows where context is currency.

Hierarchical Context

Goals inherit condo context. Sessions inherit goal context. Every agent knows what it's working on.

Real-time Status

Live dashboards show active sessions, goal progress, and system health at a glance.

Subagent Spawning

Sessions can spawn child sessions for parallel work. Parent agents stay informed.

File-backed State

All data persists as markdown files. Version control friendly. No database lock-in.

Project Isolation

Each condo is a separate workspace. Keep projects organized and contexts clean.

WebSocket API

Real-time API plus agent tools. Integrate with existing workflows. Automate everything.

Simple Integration

Designed for agent-first workflows

ClawCondos exposes clean APIs that agents can call directly. Create goals, update progress, spawn sessions — all through straightforward tool calls.
